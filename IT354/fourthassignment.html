Prototypal Inheritance

  • Functions

    • function doSomething(){}
    doSomething.prototype.foo = "bar";
    console.log( doSomething.prototype );

    This would log an object with the key foo and the value would be bar.

  • `this`, in the scope of inheritance

    • class="code">var o = {f: function() { return this.a + this.b; }};
    var p = Object.create(o);
    in this case p.a would equal 1;
