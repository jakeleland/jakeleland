Objects

  • Creating objects with `new Object()`

    • This is how to create an object in javascript with the keyword Object.

    var person = new Object();
    person.firstname = 'Jake';
    person.lastname = 'Leland';
    person.age = 22;
    console.log("My first name is " + person.firstname +"! My last name is " +person.lastname + " and I'm " +person.age +" years old!");

    This will output My first name is Jake! My last name is Leland and I'm 22 years old!

  • Bracket notation

    • Bracket notation gives you the same output as the new object() above but it's just a different way to do it which is much simpler.

    const friend = {
    firstname: 'Joe',
    lastname: 'Smith',
    age: 23
    };
    console.log("My first name is " + friend['firstname'] +"! My last name is " +friend['lastname'] + " and I'm " +friend['age'] +" years old!");

    This will output My first name is Jake! My last name is Leland and I'm 22 years old!

  • Object literals

    • Object literals is an object inside of another object this is very useful when you have a bunch of different information.

    var car = {color: 'blue', wheels: 4, engine: {cylinders: 4, size: 2}, weight: '2 tons'};
    console.log(car['engine']['size']);

    This will output 2.

  • Objects with constructors

    • Objects with constructors are when you set up the object inside of a constructor using the keyword this. An example of this is below.

    function Person(first, last, age, eye) {
    this.firstName = first;
    this.lastName = last;
    this.age = age;
    this.eyeColor = eye;
    console.log(first, last, age, eye)
    }
    Person('jake','leland',22,'blue');

  • Objects with classes

    • Creating an object with a class the most confusing but one of the most useful ways to create an object. As you can see below there is class points which is an object.

    class points {
    constructor(x, y) {
    this.x = x;
    this.y = y;
    }

    static subtract(a, b) {
    const c = a.x - b.x;
    const d = a.y - b.y;
    return c + d;
    }
    }
    const p1 = new points(5, 5);
    const p2 = new points(5, 5);
    console.log(points.subtract(p1, p2));
