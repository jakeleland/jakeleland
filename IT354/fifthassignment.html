Modules

  • Basic Modules

    • Modules are very interesting to use in javascript. They use the keyword import and export. Modules can be very useful for many different things such as different users, cars, and animals. We will first create a file called cat.js using the keywork export to send the data out of cat.js to script.js.

    export default class Cat {
    constructor(name, age) {
    this.name = name;
    this.age = age;
    }
    }
    export function printName(user) {
    console.log(`Cat's name is ${user.name}`)
    }
    export function printAge(user) {
    console.log(`Cat's age is ${user.age}`)
    }

    Next in our main script.js we will use the import keyword to import that data sent from cat.js

    import Cat from '/IT354/cat.js';
    const cat = new Cat('Rocky', 10);
    console.log(cat);

    This will display in the console the object cat with the name Rocky and age 10.

Higher Order Functions

    A higher-order function is a function that accepts a function as an argument or returns a function.

    An example that accepts a function would be :

    function sayHi() {
    return "Hi, ";
    }
    function greeting(helloMessage, name) {
    console.log(helloMessage() + name);
    }
    greeting(sayHi, "class!");

    This will return : Hi, class!

    An example of when it return a function would be :

    const returnFunction = function() {
    return function() {
    console.log("Returning a function example!");
    }
    }
    const myFunc = returnFunction();
    myFunc();

    This will return : Returning a function example!

Advanced Array Methods

    Array Methods find, findIndex, forEach, map, flat, flatMap, filter, sort, push, unshift, shift, and pop

    Examples of find, findIndex, forEach :

    const array1 = [2, 5, 8, 100, 50];
    const find = array1.find(element => element > 5);
    const findIndex = array1.findIndex(element => element > 5);
    console.log(find); // returns 8
    console.log(findIndex); // returns 2

    Example of forEach :

    const array1 = [2, 5, 8, 100, 50];
    array1.forEach(element => console.log(element)); // returns 2, 4, 8, 100, 50

    Example of map :

    const array1 = [2, 5, 8, 100, 50];
    const map = array1.map(x => x * 2); console.log(map); // returns [4, 10, 16, 200, 100]

    Example of flat :

    const flat = [1, 2, [3, 4]];
    flat.flat(); // returns [1, 2, 3, 4]

    Example of sort :

    const array1 = [2, 5, 8, 100, 50];
    array1.sort();
    console.log(array1); // [100, 2, 5, 50, 8]
