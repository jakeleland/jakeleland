Values & Types

  • Numbers

    • Numbers can be added, multiplied, divided in javascript. They can hold anything up to 2.998e8.

  • Infinity

    • Infinity is used as a numeric value that represents infinity.

  • NaN

    • NaN is used to represent a value that is not-a-number.

  • String

    • A string is a sequence of characters that can be set using double quotes "", single quotes '', or backticks ``.

Unary Operators

  • typeof

    • There are 9 different return values of typeof but the 4 major return values are number, string, boolean, and undefined. I added three buttons below that return all 4 of those types using alert. Typeof will return number if the parameters are a number such as typeof 5, a string if parameters is a string such as typeof "hello world", a boolean if parameters are true / false, and undefined if the parameters are undeclared. There are also return values such as symbol, function, bigint, null, object, and function.

  • Unary plus(+) & Unary negation(-)

    • The syntax for the Unary plus is a + before a variable, number, null, or boolean. An example of this would be +10 // 10, it also can turn a string into a number +'10' // 10 or +true will return 1. The Unary negation is a - before the variable, number, null, or boolean and is used in the same way besides it turns the number into a negative.

  • Logical Not(!)

    • The logical not operation ! returns a boolean. !true would return false and !false would return true. !'' would return true and !'hello' would return false because it is the opposite.

Hoisting

  • Hoisting

    • Hoisting is used to declare a variable at the top of the script in javascript. You have to initialize the variable at the top of the script so that the rest of the page can use it. If you do not hoist the variable at the top of the script you will not be able to use it until after it is initialized or it will return undefined..
